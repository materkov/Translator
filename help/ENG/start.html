Getting started

It's very easy to start working with program. Create new project (Program -> New project). You can change name and destination, if you want. Press OK button.

Project will be created, and new program will be added automatically. Yot can write code here.

We have simle rules. Every digit is a error (so, we have 10 error types), any other symbol is correct.

Then you can give translation command (Compile -> Projecr). In errors window you can view all errors in project.
